# PDFLibraryRnD
This is only test project which will help to build an free library which can be used to generate the PDF File. Using PdfSharp library.

The aim is to accomplish following task.

1. Split PDF
2. Merge PDF
3. IMAGES TO PDF
4. ADD PAGE NUMBER
5. ROTATE PDF
6. ADD WATERMARK
7. LOCK AND UNLOCK PDF


Other task TBD.
